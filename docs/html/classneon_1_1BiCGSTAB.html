Detailed Description

BiCGSTAB is a simple solver wrapper for the preconditioned bi-conjugate gradient stabilized solver from Eigen. This is a multithreaded solver when beneficial. The preconditioner available is incomplete Cholesky, LU and simple Jacobi.

The benefit of this solver is the ability to use a previous solution as a starting point. This is useful in time analyses as the solution is not expected to change significantly.
