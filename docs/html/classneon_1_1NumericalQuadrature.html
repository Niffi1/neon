Detailed Description

template<typename Xi, typename... Eta>
class neon::NumericalQuadrature< Xi, Eta >

NumericalQuadrature is a variadic class that defines a generic numerical quadrature class that encapsulates the coordinates, weightings and a method to perform the integration for a functor that accepts a quadrature point index.
