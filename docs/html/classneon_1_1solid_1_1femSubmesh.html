Compute the geometric stiffness matrix for in the computation solid mechanics element. The expression to be evaluated through numerical integration is the following:

\begin{align*} k_{geo} &= \sum_l^{L} B(\xi_l, \eta_l, \zeta_l)^T \sigma(l) B(\xi_l, \eta_l, \zeta_l) w(l) \end{align*}

Where B is the gradient operator in the finite element discretization
